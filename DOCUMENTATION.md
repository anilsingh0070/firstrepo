1. Project Overview

The Healthcare FAQ Bot is a simple chatbot application designed to answer common healthcare-related questions.
It provides basic information about common symptoms and health topics.

This project is developed for educational purposes and demonstrates how a rule-based chatbot works.

---

 2. Objectives

- To build a simple healthcare chatbot
- To help users get answers to basic health-related questions
- To learn Flask-based web application development

---

 3. Technologies Used

- Python
- Flask (Web Framework)
- HTML (Frontend)

---

4. System Architecture

- User enters a health-related question in the browser
- The Flask backend processes the question
- The bot matches keywords with predefined FAQs
- The relevant answer is displayed to the user
